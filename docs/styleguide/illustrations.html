Guidance and suggestions for using SVG illustrations in your project.

Illustrations

Quick comes with a beautiful set of premium SVG illustrations ready to be used in your project.

Usage

There are a few ways in which you can use these icons:

  • as static images
  • injecting them directly into the DOM via the <img> tag using the <a href=”../plugins/svg-injector.html”SVG Injector</a> plugin

Location

All the illustration files are located in assets/img/svg/illustration/ folder.
